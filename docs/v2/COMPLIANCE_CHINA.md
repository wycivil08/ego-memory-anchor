## 3. 数据跨境传输 — MVP 最大风险点

**如果使用 Vercel（美国）+ Supabase（海外）：**

| 方案 | 描述 | 适用场景 |
|------|------|---------|
| 方案 A：海外部署，仅面向海外华人 | 不触及数据跨境问题 | MVP 验证阶段 |
| 方案 B：国内云部署 | 数据不出境，完全合规 | 面向国内用户 |
| 方案 C：标准合同备案 | 与 Supabase/Vercel 签标准合同 | 中小规模跨境 |

**MVP 建议：** 先走方案 A → 验证需求 → 迁移到方案 B

## 4. 隐私政策核心条款提纲

1. 我们是谁（公司名称、联系方式）
2. 我们收集什么（账户信息、上传内容、逝者档案信息）
3. 我们不做什么（不向第三方分享、不用于AI训练、不投放广告）
4. 数据存储与安全（加密措施、访问控制RLS）
5. 您的权利（查阅、导出、删除、更正）
6. 关于逝者数据（近亲属保证条款）
7. 数据保留期限（账号存在期间永久保存，删除后30天永久删除）
