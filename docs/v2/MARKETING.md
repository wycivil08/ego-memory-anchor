# MARKETING.md — 忆锚冷启动与增长策略

## 1. 创始人故事框架

### 为什么创始人故事是最强信任杠杆

研究反复印证：grief tech 领域最成功的案例都有一个创始人亲身经历的故事。

**小公司的信任不来自品牌，来自人格。**

### 故事模板（供创始人填写）

```
[段落 1：触发]
**年____月，我失去了____。
（描述丧失的具体情境）

[段落 2：第二重丧失]
几个月后，我发现了一件比失去 TA 更让我害怕的事：
关于 TA 的记忆正在消散。
（具体描述：换了手机、某段微信语音过期了、记忆开始模糊...）

[段落 3：行动]
我开始把所有能找到的照片、视频、语音消息收集到一起。

[段落 4：产品动机]
我做忆锚，是因为我不希望任何人经历这种"第二重丧失"。

[段落 5：承诺]
忆锚不"复活"任何人，不生成 AI 对话，不合成任何内容。
我们只做一件事：守护你已有的真实记录，让它们永远不丢失。
```

## 2. 冷启动路径（0 → 50 种子用户）

### 种子用户画像

| 特征 | 描述 |
|------|------|
| 年龄 | 25-45 岁 |
| 触发时间 | 丧亲 1 天 ~ 2 年内 |
| 行为特征 | **主动在小红书/知乎搜索** "怎么保存去世亲人的照片" |
| 技术素养 | 能独立操作手机和电脑 |
| 情感状态 | 不是急性期崩溃，而是过了最初的冲击后开始"想做些什么" |

### 启动阶段（第 1-4 周）

**渠道 1：创始人的亲友圈（5-10 人）**
- 一对一私聊，亲自帮种子用户完成第一次上传

**渠道 2：丧亲互助社群（10-20 人）**
- 豆瓣"失去亲人""宠物离世"小组
- 微博 #失去亲人# #宠物去世# 话题
- 微信群（通过豆瓣/微博结识）

**绝对不做的事：**
- ❌ 在社群里发广告
- ❌ 用"我们的产品可以帮你"这种话术
- ✅ 用"我也经历过这些，我做了这个工具帮自己整理，如果你需要可以试试"

**渠道 3：小红书内容 (10-20 人)**

## 3. 内容营销矩阵

| 内容类型 | 频率 | 示例标题 |
|---------|------|---------|
| 实用教程 | 2篇/周 | "手把手教你保存微信里去世亲人的语音消息" |
| 情感故事 | 1篇/周 | "我用一个下午整理了爸爸手机里的 3000 张照片" |
| 产品展示 | 1篇/2周 | "我为爷爷做了一条从 1945 到 2023 的时间线" |
| 用户故事 | 1篇/月 | "她说：原来弟弟还保存着妈妈做饭的视频" |

**内容创作原则：**
- 有用 > 好看 > 有趣
- 标题里永远不出现"AI""数字永生""复活"
- 语气是"陪伴"不是"推销"
- 每篇内容都有一个可操作的 takeaway
