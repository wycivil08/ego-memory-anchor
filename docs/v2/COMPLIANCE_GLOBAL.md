# COMPLIANCE_GLOBAL.md — 忆锚全球合规分析

> 状态：分析报告（非法律意见）
> 适用范围：欧盟(GDPR)、美国(CCPA/CPRA)、日韩、全球通用

---

## 1. 合规要求总览

| 地区 | 法规 | 忆锚现有设计覆盖度 |
|------|------|-----------------|
| 欧盟 | GDPR | **~80%** |
| 美国（加州） | CCPA/CPRA | **~85%** |
| 韩国 | PIPA | **~75%** |
| 日本 | APPI | **~75%** |

**关键发现：忆锚的"数据主权"设计原则（导出、删除、不分析、不广告）天然满足全球主要隐私法规 80%+ 的核心要求。**

---

## 2. GDPR 数据主体权利对照

| GDPR 权利 | 忆锚实现 | 状态 |
|-----------|---------|------|
| 知情权 | 隐私政策详细说明 | ✅ |
| 访问权 | 用户可随时查看所有数据 | ✅ |
| 更正权 | 用户可编辑/删除任何素材和注释 | ✅ |
| 删除权（被遗忘权） | 一键删除账号 + 全部数据 | ✅ |
| 数据可移植性 | 一键导出 ZIP | ✅ |

---

## 4. 忆锚的天然合规优势

| 产品原则 | 对应法规精神 |
|---------|------------|
| "零合成、零篡改" | 数据最小化 + 目的限制 |
| "不做数据分析、广告" | 限制使用敏感信息 |
| "一键导出" | 数据可移植权 |
| "一键删除" | 被遗忘权 |

**产品做得越"干净"，合规成本越低。这是一种正循环。**
