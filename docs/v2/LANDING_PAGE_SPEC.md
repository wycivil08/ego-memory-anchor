## 4. 性能要求

| 指标 | 目标 |
|------|------|
| First Contentful Paint | < 1.0s |
| Largest Contentful Paint | < 2.0s |
| Lighthouse Performance | > 90 |
| Lighthouse SEO | > 95 |
| Lighthouse Accessibility | > 95 |

**实现要点：**
- Hero 插画用 SVG 或 CSS 渐变，不用大图
- 不引入第三方分析/追踪脚本
- 使用 next/font 加载 Inter
- 所有图片使用 Next.js Image 组件
